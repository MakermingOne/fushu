# macOS 屏保文件

提供了两种方式在 macOS 上使用负鼠表情包作为屏保：

---

## 方式一：系统屏保（推荐）

文件：`Fushu.saver.zip`

这是**真正的 macOS 屏保**，可在系统设置中选择。**完全离线运行**，无需网络。

### 安装步骤

1. 解压 `Fushu.saver.zip`
2. 双击 `Fushu.saver`，系统会提示安装
3. 选择「为这台 Mac 的所有用户安装」或「仅为我安装」
4. 前往 **系统设置 → 锁定屏幕 → 屏幕保护程序**
5. 在左侧列表中找到「负鼠屏保」，点击选择

> 如果提示「无法打开，因为无法验证开发者」，请右键点击 `Fushu.saver`，选择「打开」。

---

## 方式二：独立 App

文件：`FushuScreenSaver.app.zip`

这是一个**全屏 App**，双击即可运行，鼠标移动自动退出。**完全离线运行**，无需网络。

### 使用步骤

1. 解压 `FushuScreenSaver.app.zip`
2. 将 `FushuScreenSaver.app` 拖入「应用程序」文件夹
3. 双击运行，全屏显示负鼠表情包
4. **移动鼠标或按任意键即可退出**

### 设置为开机启动

前往 **系统设置 → 通用 → 登录项**，将 `FushuScreenSaver.app` 添加到「登录时打开」。

---

## 常见问题

**Q: 运行时报错「无法验证开发者」**
> A: 右键点击文件 → 选择「打开」→ 在弹窗中点击「打开」。

**Q: 屏保是黑屏？**
> A: 旧版本依赖网络加载网页，新版已改为**完全离线**，内置了所有资源，不会再黑屏。

**Q: 需要联网吗？**
> A: **不需要**。新版屏保将网页和资源（atlas.jpg）打包在内部，无需任何网络连接。
